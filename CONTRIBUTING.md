# Contributing to RelayNet Core

Thank you for your interest in contributing to RelayNet Core.

## Contribution Guidelines

We welcome contributions in the form of:

- Bug reports
- Feature discussions
- Documentation improvements
- Code contributions via pull requests

All contributions must be discussed through issues before submitting major changes.

## Code of Conduct

All contributors are expected to act professionally and respectfully.
Harassment or abusive behavior will not be tolerated.

## Getting Started

Please refer to the README and documentation in the `docs/` directory
before submitting your first contribution.
